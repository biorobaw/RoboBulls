Detailed Description

General-purpose skill to kick the ball to a point.

Author
JamesW

A simplified, optimized version of KTP for Omni robots. The ability the track a moving target is preserved with the pointer constructor. The robot is asked to first move behind the ball while rotating to face the ball. Then the Robot moves forwards, and finally kicks the ball when close enough.

KTPO supports both a static point and variable point constructors.

Examples:
example_kicktopointomni.cpp.
