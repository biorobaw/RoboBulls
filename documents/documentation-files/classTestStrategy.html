Detailed Description

Class to test behaviors and other code.

Not part of the main game, TestStrategy exists as a sandbox to test code. To use, change REFBOX_LISTEN_ENABLED to 0 in include/config/communication.h. By doing this, StrategyController will always select and run TestStrategy instead of the strategy given by the RefBox.
