Detailed Description

Defines comparison predicates and information querying functions.

Comparisons is a framework for extendable, flexible predicate logic functionality. Comparisons defines a set of commonly-used predicates and comparison functions to query information on containers of Robot* or Point. These functions are meant to be used with the max_element and min_element STL algorithms, or standalone.

A commonly-used query we need, for example, is to find the robot closest to the ball. Using Comparisons:
Robot* best = Comparisons::distanceBall().minMyTeam();

It can also be chained to provide complex queries easily:
//Robot on My Team, not the goalie, closest to the robot farthest from the ball
Robot* r = Comparisons::distance(*Comparisons::distanceBall().maxMyTeam()).ignoreID(GOALIE_ID).minMyTeam();

Comparisons defines many object functors like these for various min/max comparisons on containers, along with ignore functions to not consider certain IDs in these comparisons.
