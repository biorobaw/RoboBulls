bool DefenceArea::contains ( const Point &  p,
const float  tolerance

Modified contains() function that returns true if the input Point is within tolerance distance from the defence area. This is useful to see if any part of a robot is within the defence area since only the centre of the robot is checked.

void DefenceArea::expelPoint ( Point &  point)

Given a Point (by reference) that is inside the defence area, modifies it such that it is outside the defence area while being as close as possible to the original point. Useful for path-planning.
