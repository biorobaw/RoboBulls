Detailed Description

Behavior to rotate around a stationary point.

Author
James W

This behavior is omni-only. It allows a robot to approach a point from any angle, and rotate on a circle, facing the center, to face a final direction of faceAngle. Additionally, the dist argument specifies the dist to stay from where when rotating. isFinished() returns true when the robot is near the target point on the circle.
